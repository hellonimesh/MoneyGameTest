Install Visual studio 2019 community edition and select unitTestTemplate(.net).

The following packages need to be installed from Nuget paackage manager:

1) chromedriver
2) Nunit3
3) Nunit3 test adaptor
4) Specflow
5) specflow for visual studio 2019 as an extension
6) DotNetSeleniumExtras.PageObjects
7) DotNetSeleniumExtras.PageObjects.Core
8) Specflow.Nunit
9) Selenium Support
10) specflow tools.msbuild.generation
